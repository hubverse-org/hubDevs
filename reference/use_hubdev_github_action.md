# Hubverse Developer GitHub Action setup

Sets up common hubverse developer continuous integration (CI) workflows
for hubverse packages hosted on GitHub using [GitHub
Actions](https://github.com/features/actions). Available actions are
hosted in repository
[hubverse-org/hubverse-developer-actions](https://github.com/hubverse-org/hubverse-developer-actions)

## Usage

``` r
use_hubdev_github_action(name)
```

## Arguments

- name:

  Name of workflow, i.e. the name of one of the [hubverse developer
  action
  repository](https://github.com/hubverse-org/hubverse-developer-actions)
  directories containing a GitHub Action workflow `.yaml` file.
